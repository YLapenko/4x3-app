EN 
-----
This is a small PWA for adding frames to vertical (or horizontal) photos to bring them to a 4:3 aspect ratio before posting to Instagram.

RU
----
Это небольшое PWA-приложение для добавления рамок к вертикальным (но можно и к горизонтальным) фотографиям для приведения фотографии к формату 4х3 перед публикацией в Instagram.
